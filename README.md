# Brave-Trigger
Simulate user activity for Windows only. Can trigger and click on Brave Ads.


## Installation
```
pip install pydirectinput
or
pip3 install pydirectinput
```
## Launch
```
git clone https://github.com/mikedigriz/Brave-Trigger.git
cd Brave-Trigger
python3 simulate.py
or
python simulate.py
```
